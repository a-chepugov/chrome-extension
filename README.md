# Помогайка

## установка

- открыть страницу расширений [chrome://extensions](chrome://extensions)
- включить режим разработчика
- загрузить расширение

## функционал

- сохранение в хранилище:
  - локальное
	- синхронизируемое
- распознание голоса:
  - в поле ввода
  - в хранилище
-	поиск:
  - найти определние
  - текущая страница в кеше
  - поиск в пределах текущего сайта
